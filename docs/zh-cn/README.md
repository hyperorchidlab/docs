### 安装矿池和矿机

+ 上传可执行程序到Linux服务器,这里使用CentOS作为例子

!>**强烈建议矿池和矿机的服务器独立并且分开，使用同一台服务器会降低矿池和矿机的响应和稳定**

```console
$ scp Pool root@yourserverip:/root/pool
```

```console
$ scp HOP root@yourserverip:/root/hop
```
